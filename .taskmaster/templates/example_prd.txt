# Project Name: 우리 아기 하루 (Baby Emotion Video Log)
# Version: 1.0
# Last Updated: 2025-08-05
# Author: 쭈니

## 📌 개요
우리 아기 하루는 육아 중인 부모를 위한 감정 기반 영상 기록 앱입니다.
AI 분석을 통해 영상 속 아기의 감정을 자동 분류하고, 일자별 감정 일기를 작성할 수 있으며, 가족과 감정 기록을 쉽게 공유할 수 있는 플랫폼입니다.

---

## 📱 주요 화면 및 기능

### 1. 홈화면 (감정 요약)
- 오늘 감정 통계 (웃음, 울음, 졸림 등)
- 최근 영상 리스트
- 영상 추가 버튼

### 2. 영상 업로드 화면
- 영상 파일 선택 or 녹화
- 감정 분석 API 연동
- 감정 결과 및 태그 표시
- 저장 / 다시 선택 기능

### 3. 영상 상세 보기
- 영상 재생
- 감정 분석 결과 표시
- 추천 태그 및 수동 태그
- 가족에게 공유

### 4. 감정 일기장 (월별)
- 달력형 UI
- 날짜별 감정 이모지 표시
- 클릭 시 해당 날짜 영상 목록

### 5. 가족 공유
- 가족 초대 / 수락
- 공유 범위 설정
- 공유된 영상 목록

### 6. 설정 화면
- 사용자 정보 수정
- 클라우드 용량 확인
- 로그아웃
- 피드백 전송

---

## 🔧 기술 스택

### 프론트엔드
- Flutter + stacked-cli 구조
- State 관리: stacked ViewModel
- **모든 UI는 재사용 가능한 컴포넌트로 분리**하여 구현합니다.
  - 이유: 쭈니가 사용하는 기본 템플릿이 있으며, 해당 템플릿에 쉽게 통합할 수 있도록 하기 위함입니다.

### 백엔드
- 백엔드는 쭈니가 직접 개발할 예정입니다.
- 현재 문서에 명시된 백엔드 기술 예시는 참고용입니다.
- Supabase (Auth, Storage)는 필요 시 사용

### 감정 분석
- Microsoft Azure Face API (표정 기반)
- Google Mediapipe (보조)

### 기타
- 영상 처리: ffmpeg (서버 or 플랫폼 의존)
- 공유: 초대 링크 기반 / Firebase Dynamic Links 고려

---

## 🧠 감정 분석 흐름

1. 영상 업로드 시 썸네일 추출 or 프레임 중 일부 캡처
2. 캡처 이미지 → Face API 요청
3. 감정 결과 → 확률 기반으로 최고 감정 선정
4. 분석 결과 저장 및 사용자에게 표시

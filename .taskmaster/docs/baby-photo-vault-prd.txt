# Baby Photo Vault - 혁신적 아기 사진 관리 앱 PRD

## 1. 프로젝트 개요

### 1.1 제품명
**Baby Photo Vault** - AI 기반 아기 성장 기록 및 가족 공유 플랫폼

### 1.2 핵심 가치 제안
- **로컬 우선 보안 모델**: 개인정보 보호를 최우선으로 하는 로컬 암호화 저장
- **AI 자동 큐레이션**: 매일 촬영된 사진을 자동으로 선별하고 의미 있는 순간을 감지
- **가족 권한 기반 공유**: 세밀한 권한 설정으로 안전한 가족 간 공유
- **실물 연동**: 디지털 추억을 물리적 상품(포토북, 기념품)으로 변환

## 2. GI (Genius Insight) 분석 결과

### 2.1 관찰 및 점수화
- **O (관찰의 깊이)**: 9/10 - 신생아 사진의 높은 빈도와 감정적 가치
- **C (연결의 독창성)**: 9/10 - 사진×성장데이터×건강정보×실물상품의 이종 결합
- **P (패턴 인식)**: 8/10 - 첫 걸음, 첫 이, 수면 패턴 등 반복되는 성장 패턴
- **S (종합적 사고)**: 8/10 - 기술·심리·상품화 관점의 통합적 설계
- **A (고정관념)**: 1/10 - 기존 클라우드 중심 앱의 가정 배제
- **B (편향)**: 1/10 - 가족 중심 프라이버시 우선 관점

**GI 점수**: 2592 (높은 천재적 통찰 가능성)

### 2.2 핵심 통찰
- 단순 사진 저장을 넘어 "시간(성장)·문맥(이벤트)·사람(가족 역할)" 연결
- 프라이버시·오프라인 우선 모델이 부모 신뢰와 지불 의사 유발
- 자동화(ML) + 큐레이션(부모 편집) + 실물 연동의 3축 조합이 최적

## 3. IS (Innovative Solution) 기반 핵심 기능

### 3.1 우선순위 1: 타임라인 스토리북
- **기능**: 매주/매월 베스트샷 자동 추출 → 자막+음악이 포함된 1분 타임랩스 생성
- **차별성**: 부모가 일일 사진을 손대지 않아도 성장 스토리 소비 가능
- **수익모델**: 월 구독(HD 비디오/PDF 다운로드), 물리적 포토북 주문

### 3.2 우선순위 2: 가족 권한 Vault
- **기능**: 사진별 세부 공유 권한 설정(엄마·아빠·조부모·외가 등)
- **차별성**: 개인정보·초상권 중시하는 부모층 강타
- **수익모델**: B2B 버전(유아원·병원), 프리미엄 보안 기능

### 3.3 우선순위 3: 로컬-퍼스트 백업
- **기능**: 기기 내 암호화 저장 + 선택적 분산/클라우드 동기화
- **차별성**: 보안 우선 모델로 신뢰 확보
- **수익모델**: 프리미엄 복구 기능, 보안 감사 서비스

### 3.4 우선순위 4: 감성 캡션 & AI 편집자
- **기능**: 사진 분위기 분석 → 감성 캡션 + 자동 음악 추천
- **차별성**: 부모가 스토리텔링에 고심할 필요 없음
- **수익모델**: 프리미엄 음악/캡션 템플릿, 다국어 번역

### 3.5 우선순위 5: 첫(Firsts) 자동 감지
- **기능**: '첫 미소, 첫 이유식, 첫 걸음' 등 ML 기반 자동 태깅
- **차별성**: 부모가 의식하지 못한 '첫' 순간도 캡처
- **수익모델**: '첫 컬렉션' 포장 상품, 자동 앨범 판매

## 4. 기술 스택 및 아키텍처

### 4.1 프론트엔드
- **Flutter**: 멀티플랫폼 개발 (iOS, Android, Web)
- **상태관리**: Riverpod 또는 Bloc
- **UI**: Material Design 3 + 커스텀 테마

### 4.2 백엔드 및 저장소
- **로컬 저장**: SQLite + 암호화 (SQLCipher)
- **클라우드**: 선택적 Supabase 연동
- **파일 저장**: 로컬 우선 + 암호화된 동기화

### 4.3 AI/ML 서비스
- **온디바이스 ML**: TensorFlow Lite (표정 감지, 객체 인식)
- **클라우드 AI**: Google Vision API (고급 이미지 분석)
- **비디오 처리**: FFmpeg (썸네일, 압축)

### 4.4 보안 및 인증
- **암호화**: AES-256 (로컬), TLS 1.3 (전송)
- **인증**: Supabase Auth (이메일, 소셜 로그인)
- **권한 관리**: RBAC (Role-Based Access Control)

## 5. 사용자 여정 및 핵심 시나리오

### 5.1 신규 사용자 온보딩
1. 앱 설치 및 권한 설정 (카메라, 저장소)
2. 가족 구성원 등록 및 권한 설정
3. 첫 사진 업로드 및 AI 분석 체험
4. 타임라인 스토리북 미리보기

### 5.2 일상 사용 시나리오
1. **사진 촬영**: 자동으로 앱에 저장, AI 분석 시작
2. **자동 큐레이션**: 의미 있는 순간 자동 선별 및 태깅
3. **가족 공유**: 권한에 따른 선택적 공유
4. **스토리 생성**: 주간/월간 자동 스토리북 생성

### 5.3 특별한 순간 관리
1. **첫 순간 감지**: AI가 자동으로 '첫' 순간 감지 및 알림
2. **기념일 관리**: 100일, 1년 등 자동 알림 및 템플릿 제공
3. **실물 주문**: 포토북, 캔버스 등 즉시 주문 가능
